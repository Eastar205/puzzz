# 게임 기획 컨셉 - Puzzz (가제)

작성일: 2026-07-16

## 1. 개요
- 장르: 퍼즐형 캐릭터 디펜스 (Connect Puzzle + Auto-Battler Defense)
- 플랫폼: iOS / Android (동시 출시)
- 엔진: Unity (2D + Box2D 기반 물리)
- 핵심 타겟: 캐주얼한 조작감과 RPG 캐릭터 육성/전투 쾌감을 동시에 즐기고 싶은 유저
- 핵심 재미 요소: 퍼즐을 길게 이을 때의 '손맛'과, 모인 마나로 적을 쓸어버리는 '스킬 연계'의 쾌감

## 2. 화면 구성
화면은 상단(전투)과 하단(퍼즐) 두 영역으로 나뉘며, 하단 퍼즐 조작이 실시간으로 상단 전투에 개입한다.

- 상단 - 디펜스 구역: 4명의 아군 캐릭터가 진형을 갖추고 몰려오는 몬스터 웨이브와 자동 전투 진행 (기본 공격 자동)
- 하단 - 퍼즐 구역: 7x7 격자에 5종류의 구슬이 배치된 드래그 연결 조작 패널

## 3. 코어 루프
구슬 연결 및 파괴 → 해당 속성 캐릭터 마나 충전 / 버프 획득 → 상단 캐릭터의 스킬 발동 → 웨이브 클리어 및 보상 획득

- 플레이어가 손가락을 누른 채 드래그하면 상하좌우로 이어진 같은 색 구슬이 체인으로 연결됨
  - 바로 이전 칸으로 되돌아가면 체인이 한 칸 줄어듦 (실수 정정 가능)
- 손을 떼면 체인이 확정되어 제거되고, 지워진 개수만큼 위에서 새 구슬이 채워지며 낙하 (끝없이 이어지는 리필형 퍼즐)
- 길게 연결할수록 마나 충전량에 보너스 배율이 붙어 유저의 실력이 전투 결과에 직결됨

## 4. 구슬(Orb) 시스템

| 구슬 종류 | 등장 확률 | 효과 |
| - | - | - |
| 속성 볼 (적/청/녹/황 4종) | 각각 약 23.75% (균등) | 파괴한 색과 동일한 속성 캐릭터의 마나 충전. 마나 100% 도달 시 고유 스킬 발동 |
| 특수 볼 (보라색) | 5% (희귀) | 마나 대신, 연결 개수에 비례한 시간 동안 아군 전체 공격 속도 증폭 버프 |

- 마나 충전량: `기본치 x 체인 길이 x 체인 보너스 배율` (체인 4 이상 x1.5, 7 이상 x2)
- 보라색 버프 지속시간: 기본 5초(2연결 기준) + 연결이 늘어날 때마다 1.5초씩 추가

## 5. 캐릭터 구성 (1차 프로토타입)
덱빌딩 없이 4속성 고정 1명씩 배치, 이후 버전에서 편성/덱빌딩 확장 예정

| 속성 | 캐릭터 | 스킬 효과 |
| - | - | - |
| 빨강 | 전사 | 전방 몬스터에게 강한 단일 피해 |
| 파랑 | 마법사 | 전방 몬스터에게 강한 단일 피해 |
| 초록 | 궁수 | 전방 몬스터에게 강한 단일 피해 |
| 노랑 | 힐러 | 몬스터 공격이 아닌, 아군 기지 체력 회복 |

- 스킬 발동 방식은 자동/수동 중 선택 가능하도록 캐릭터별 설정 제공
  - 자동: 마나가 차면 즉시 시전 (캐주얼)
  - 수동: 마나가 차면 대기 상태가 되고, 유저가 터치해야 시전 ("지금 쓸까, 더 모을까" 자원 관리 전략성)

## 6. 전투 구조
- 몬스터는 웨이브 단위로 순차 스폰되어 기지를 향해 이동
- 몬스터가 기지에 도달하면 기지 체력 감소, 기지 체력이 0이 되면 실패
- 캐릭터는 기본 공격을 자동으로 반복하며, 스킬은 마나 충전으로 발동
- 몬스터를 모두 처치하면 웨이브 클리어

## 7. 차별화 포인트
1. **보라색 볼의 전략적 활용 (조커 카드)**: 등장 확률이 낮아 모으기 어렵지만, 위기 상황에 길게 터뜨려 순간 폭딜(공격 속도 폭발)을 넣는 핵심 전략 요소. 바로 터뜨리지 않고 킵해두는 두뇌 플레이 유도
2. **길게 잇기(Long Chain)의 보상 심리**: 짧게 여러 번보다 한 번에 길게 연결했을 때 마나 충전 보너스가 커서, 유저가 퍼즐판을 넓게 보도록 유도. "지금 스킬을 쓸 것인가, 다음 웨이브까지 모을 것인가" 자원 관리의 재미
3. **캐릭터 속성 조합의 묘미** (확장 예정): 특정 속성 캐릭터를 중복 배치해 특정 색상 볼 효율을 극대화하거나, 4속성 균형 편성을 하는 덱빌딩 재미 (1차 프로토타입 이후 도입)

## 8. 로드맵
- **1차 (현재)**: 퍼즐-마나-스킬-웨이브 핵심 루프만 검증. 피버타임 제외, 캐릭터 4속성 고정
- **2차**: 피버타임(콤보/점수 도달 시 전체 색상 통일 또는 보라볼 드랍률 급상승), 보라볼 킵 전략 강화, 캐릭터 덱빌딩

## 9. 수익 모델 (추후 확정)
- 웨이브 실패 시: 리워드 광고 시청으로 기지 체력 일부 회복 후 재도전
- IAP: 캐릭터 스킨, 시즌패스, 재화 패키지

## 10. 아트 / 사운드 방향
- 아트: 미니멀 파스텔톤, 젤리/캔디 느낌 구슬 + 코믹한 SD 캐릭터
- 사운드: 경쾌한 팝 사운드, 스킬 발동 시 임팩트 있는 이펙트음

## 11. 기술 스택
- Unity 2D, Box2D 기반 물리 시뮬레이션
- C#
- iOS / Android 동시 빌드
